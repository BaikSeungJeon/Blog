### Scope
[이전에 작성한 글](https://github.com/BaikSeungJeon/TIL/blob/main/2022/10/2022%2010%2022%20Scope%20%26%20Tree%20Shaking.md)

```js
var x = 'global';

function foo () {
  var x = 'function scope';
  console.log(x);
}

foo(); // function scope
console.log(x); // global
```
